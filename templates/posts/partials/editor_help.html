Formatting Guide

Use Markdown or HTML to format your post

Headings

# Heading 1

## Heading 2

### Heading 3

Text Styles

**bold text**

*italic text*

__underlined text__

***bold and italic***

Lists

Bullet points:

- Item one

- Item two

Numbered lists:

1. First item

2. Second item

Images

![Alt text](https://example.com/image.jpg)

Or use HTML:

<img src="url" alt="description">

Videos

YouTube (just paste the URL):

https://youtube.com/watch?v=VIDEO_ID

Or short link:

https://youtu.be/VIDEO_ID

Code

Inline code:

`code here`

Code block:

```

your code here

```

Links & Quotes

[Link text](https://example.com)

> This is a blockquote

--- (horizontal line)

Tip: Images and videos added to your post will automatically appear in the Media Gallery!
